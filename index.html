Fruno App

This is a Go-based web application that scrapes data from the Algolia API and the ProtonDB API. It provides a simple API endpoint to retrieve information about games, including their name, objectID, and Linux compatibility tier.

How to Use

To use the scraper, make a GET request to the "/scrape" endpoint with a "name" query parameter. For example: 

http://localhost:8080/scrape?name=Far Cry 4

This will return a JSON response containing the name, objectID, and Linux compatibility tier for the first game that matches the search query.

Running the App

To run the app locally, follow these steps:

  1. Clone the repository: git clone https://github.com/theofficialpeter/Fruno.git
  2. Navigate to the project directory: cd Fruno
  3. Build and run the app: go run scraper.go

The app will start running on http://localhost:8080.
